# Pacific Pulse AI

## Project Overview
Frontend landing page and demo for Pacific Pulse AI services. A marketing website showcasing AI capabilities with smooth animations.

## Tech Stack
- **Framework:** Next.js 16 (App Router) with React 19
- **Database:** None (frontend only)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## App Structure
```
app/
├── page.tsx           # Landing page
├── layout.tsx         # Root layout
└── globals.css        # Global styles
```

## Key Features
- Animated landing page
- Service showcases
- Contact sections
- Responsive design

## Important Notes
- This is a frontend-only project (no database)
- Uses Tailwind CSS 4 (consistent with other projects)
- No authentication required
- Focus on smooth animations with Framer Motion

## Development
```bash
npm run dev    # Start development server
npm run build  # Build for production
npm run lint   # Run linting
```
